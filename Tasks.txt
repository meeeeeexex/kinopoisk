1. Запрос для поиска по фильму,  найденные фильмы по ключевому слову
2. Страница фильма - список фильмов, фильтрация фильмов по жанру, режисеру, актеру, кинотеатру
3. Страница актера - список актеров, фильтрация актеров по фильму, жанру (фильмография)
4. Страница режисера - список режисеров, фильтрация режисеров по фильму, жанру (фильмография)
5. Страница кинотеатра - список кинотеатров, фильтрация кинотеатров по фильму
6. Запрос обзоров - список рецензий по фильму, количество рецензий по фильму
7. Создание вопросов, например: «Является ли жанр фильма [жанр]?», «Снимался ли [актер] в данном фильме?»,
«Фильм был снят в [страна]?»
